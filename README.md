## 🎯 Projects Overview & Architecture

### 🧮 Task-1 Calculator
A fully functional, responsive web-based calculator capable of performing standard arithmetic calculations.
* **Key Features:** Grid-based button interface, modern neumorphic/glassmorphic look, and real-time execution of operational logic.
* **Tech Stack:** HTML5, CSS3, JavaScript (ES6+ Arithmetic Evaluation).

### 📜 Task-2 Tribute Page
A beautifully styled static webpage dedicated to honoring a prominent personality, focusing on semantic layout structure.
* **Key Features:** High-quality image integration, a detailed timeline of achievements, elegant layout typography, and clean text alignments.
* **Tech Stack:** HTML5, CSS3 (Custom styling and layouts).

### 📝 Task-3 Todo WebApp
An interactive CRUD (Create, Read, Update, Delete) task management application designed for tracking daily objectives.
* **Key Features:** Dynamic task lists addition, mark-as-completed functionality, interactive UI elements, and data persistence simulation.
* **Tech Stack:** HTML5, CSS3, JavaScript (DOM Manipulation & Event Handlers).
